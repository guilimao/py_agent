# PyAgent

一个使用 Python 构建的命令行 LLM 代理示例。

## 概述

PyAgent 是一个基于 Python 实现的命令行 LLM（大型语言模型）代理。该项目展示了如何构建一个交互式代理，能够处理自然语言命令并执行各种任务。

## 功能

- 交互式命令行界面
- 自然语言处理能力
- 可扩展架构，支持添加新功能

## 使用方法

```bash
# python main.py
```
